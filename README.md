## 📋 Описание

**MeshChat** — Android-приложение для обмена зашифрованными сообщениями через mesh-сеть на базе Wi-Fi Direct (2.4 ГГц). Не требует интернета или Wi-Fi роутера — устройства соединяются напрямую друг с другом.

### Ключевые особенности

- 🔐 **Сквозное шифрование** — RSA-2048 для обмена ключами + AES-256-GCM для шифрования сообщений
- 📡 **Wi-Fi Direct (2.4 ГГц)** — прямое P2P-соединение между устройствами
- 🏦 **Распределённый банк сообщений** — все узлы хранят зашифрованные сообщения до прочтения адресатом
- 🧹 **Автоматическая очистка** — после прочтения сообщение удаляется у промежуточных узлов, но сохраняется у отправителя и получателя
- 🎨 **Material Design 3** — современный тёмный интерфейс на Jetpack Compose

## 🔒 Протокол шифрования

### Схема работы

1. **Генерация ключей** — При первом запуске каждое устройство генерирует пару ключей RSA-2048 (хранятся в Android Keystore)
2. **Обмен ключами** — При обнаружении устройств публичные ключи RSA обмениваются через Wi-Fi Direct
3. **Шифрование сообщения**:
   - Генерируется случайный AES-256 ключ
   - Сообщение шифруется AES-256-GCM (аутентифицированное шифрование)
   - AES-ключ шифруется публичным RSA-ключом получателя (RSA-OAEP SHA-256)
4. **Передача** — В mesh-сеть отправляется: зашифрованное сообщение + зашифрованный AES-ключ + IV
5. **Хранение в банке** — Все узлы хранят зашифрованные данные, не имея возможности их прочитать
6. **Расшифровка** — Только получатель расшифровывает AES-ключ своим приватным RSA-ключом
7. **Очистка** — После прочтения рассылается `ReadReceipt`, промежуточные узлы удаляют сообщение

### Безопасность

| Компонент | Алгоритм | Назначение |
|-----------|----------|------------|
| Обмен ключами | RSA-2048 OAEP (SHA-256) | Шифрование AES-ключа |
| Шифрование сообщений | AES-256-GCM | Конфиденциальность + аутентичность |
| ID узла | SHA-256 (от публичного ключа) | Уникальная идентификация |
| Хранение ключей | Android Keystore | Защита приватного ключа |

## 🛠 Технологии

- **Kotlin** — основной язык
- **Jetpack Compose** — декларативный UI (Material 3)
- **Wi-Fi Direct API** — mesh-сеть (2.4 ГГц P2P)
- **Room** — локальная база данных
- **Android Keystore** — безопасное хранение ключей
- **MVVM** — архитектурный паттерн
- **Coroutines + Flow** — асинхронность

## 🚀 Сборка и запуск

### Требования
- Android Studio Hedgehog (2023.1.1) или новее
- Android SDK 34
- Минимальная версия Android: 8.0 (API 26)
- Два или более Android-устройства с Wi-Fi Direct

### Сборка
```bash
cd MeshChat
./gradlew assembleDebug
```

### Установка
```bash
./gradlew installDebug
```

### Тесты
```bash
./gradlew test
```
